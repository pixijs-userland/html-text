Class: HTMLText

HTMLText

Text display object that support multi-style HTML text.

new HTMLText (text, style)

Name Type Attributes Default Description
text string <optional>
 ''

Text contents
style HTMLTextStyle | PIXI.TextStyle | PIXI.ITextStyle <optional>

Style setting to use. Strongly recommend using an HTMLTextStyle object. Providing a PIXI.TextStyle will convert the TextStyle to an HTMLTextStyle and will no longer be linked.
See:
  • PIXI.Sprite
  • PIXI.TextStyle

height number

Get the height in pixels.

maxHeight number

The maximum height in rendered pixels that the content can be, any larger will be hidden

maxWidth number

The maximum width in rendered pixels that the content can be, any larger will be hidden

resolution number

The resolution / device pixel ratio of the canvas. This is set to automatically match the renderer resolution by default, but can be overridden by setting manually.

Default Value:
  • 1

source HTMLImageElement

The raw image element that is rendered under-the-hood.

style HTMLTextStyle

The base style to render with text.

text string

Contents of text. This can be HTML text and include tags.

Example
const text = new HTMLText('This is a <em>styled</em> text!');

width number

Get the width in pixels.

measureText (overrides) PIXI.ISize

Calculate the size of the output text without actually drawing it. This includes the padding in the style object. This can be used as a fast-pass to do things like text-fitting.

Name Type Attributes Description
overrides object <optional>

Overrides for the text, style, and resolution.
overrides.text string <optional>

The text to measure, if not specified, the current text is used.
overrides.style HTMLTextStyle <optional>

The style to measure, if not specified, the current style is used.
overrides.resolution number <optional>

The resolution to measure, if not specified, the current resolution is used.
Returns:
Type Description
PIXI.ISize Width and height of the measured text.
